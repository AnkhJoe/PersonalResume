Project Details 1

Project content: deployed in Huawei cloud using Huawei OpenEuler through ISCSI technology to remotely mount the storage space in the server to the local, to realize storage space sharing

Responsible for: Linux scripting, OpenEuler cloud deployment, OpenGauss database allocation.

Project results: Solves the problem of short-term rental of storage services for non-professional users. Adopting the P2P sharing model to make fuller use of resources, innovating and integrating some of Huawei's storage technologies, the project has stronger adaptability and better application value.
